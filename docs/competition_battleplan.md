# 抖医 TiktokRx — 竞赛策略

## 核心卖点

### 1. 数据驱动（非纯 LLM）
- 8880+ 条真实数据训练
- Spearman 相关 + 线性回归 + K-Means 聚类
- 6 品类差异化基线权重
- Model A 纯数学评分（< 50ms，无 LLM 调用）

### 2. 多Agent辩论（非单次调用）
- 6 个专业化 Agent 并行诊断
- 三轮辩论交叉验证
- 模拟专家会诊机制
- 有效减少单模型偏差

### 3. 完整闭环（非只给建议）
- 自动生成优化方案
- 即时重新评分
- 最高分方案推荐
- 评论区预测

### 4. 多模态支持
- 视频链接 / 截图上传 / 纯文字 / BGM 名称
- MiMo-v2-Omni 视频理解
- Whisper STT 口播转写
- OCR 文字提取

## 应对评委可能的问题

**Q: 和直接用 ChatGPT 诊断有什么区别？**
A: 三个核心差异：
1. 数据驱动 — 我们用 8880+ 条真实数据训练了品类差异化基线，不是通用 LLM 打分
2. 多Agent辩论 — 6 个专家三轮辩论交叉验证，不是单次调用
3. 完整闭环 — 自动生成优化方案并重新评分，不是只给建议

**Q: 数据来源是否合规？**
A: 所有数据通过公开渠道采集，已脱敏处理，不包含任何个人隐私信息。仅用于研究目的。

**Q: 评分准确率如何？**
A: Model A 基于 8880+ 条真实数据的回归分析，具有统计学意义。多Agent辩论机制进一步提高了诊断质量。

**Q: 支持哪些内容类型？**
A: 短视频（6维）、图文（5维）、纯文字（5维）、BGM分析（独立评分），覆盖抖音主要内容形态。
